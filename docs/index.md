# VM Setup Tool

Welcome to the documentation for **VM Tool**.

## Installation

```bash
pip install vm-tool
```

## Features

- **Automated Setup**: Configure VMs using Ansible playbooks.
- **Zero-Touch Deployment**: Push code, configure Docker, and deploy with one command.
- **Observability**: One-click Prometheus & Grafana stack.
- **CI/CD Generator**: Create GitHub Actions pipelines interactively.ows.
- **SSH Management**: Auto-generate keys and configure `~/.ssh/config`.
- **Cloud Ready**: Dynamic inventory for cloud hosts.
