# VM Setup Tool

Welcome to the documentation for **VM Tool**.

## Installation

```bash
pip install vm-tool
```

## Features

- **Automated Setup**: Configure VMs using Ansible playbooks.
- **Infrastructure Provisioning**: Terraform integration for cloud providers.
- **Kubernetes Ready**: One-click K3s cluster setup.
- **Observability**: Instant Prometheus & Grafana deployment.
- **CI/CD Pipelines**: Auto-generate GitHub Actions workflows.
- **SSH Management**: Auto-generate keys and configure `~/.ssh/config`.
- **Cloud Ready**: Dynamic inventory for cloud hosts.
